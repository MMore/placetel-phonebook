placetel-phonebook
==================

Generates a phonebook via the Placetel API.


Configuration
--------------

* Open phonebook.rb
* Set...
  * placetel_api_key
  * title
* Done!


Run
-------------

Just start the application with `ruby phonebook.rb` and open **http://localhost:4567** in a browser. Or use something
like passenger or unicorn to serve it.
